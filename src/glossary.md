# Glossary

Define a term once as `term :: definition`. Every occurrence in any frame's prose
becomes a hover card (the first occurrence per frame). Delete these and write
your own — they are just examples.

FLIP :: First-Last-Invert-Play — animate layout change by measuring before/after and tweening the difference.
frame :: One slide; everything under a single `## ` heading.
lens :: A panel that renders the frame (code, graph, or file tree) and animates by the same keyed rule.
