# History

## the repo, a few commits back

A fourth lens: git history, read from real `git log`. This is the branch a few commits back. Newest commit on top, HEAD marked.

git: ../.. HEAD~4 5

## a few commits later

Advance the revision and the timeline animates: shared commits slide down, newer ones fade in at the top, HEAD moves with them. Keyed by sha — the same FLIP rule as the file tree.

git: ../.. HEAD~1 5

## up to now

The tip of the branch. Every lens — code, graph, filesystem, git — animates by matching keys and sliding what moved. See [[04-filesystem]] for the same rule on a file tree.

git: ../.. HEAD 5
